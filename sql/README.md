# SQL Scripts

This folder contains the SQL files used to build and practice the Little Lemon database.

## Script Order

Run the files in this order:

1. `00_setup.sql` - Creates the database
2. `01_schema.sql` - Creates the tables
3. `02_seed.sql` - Inserts sample data
4. `03_core_queries.sql` - Basic query examples
5. `04_structure_and_updates.sql` - Update and structure tasks
6. `05_subqueries_and_views.sql` - Subqueries and views
7. `06_procedures_and_strings.sql` - Stored procedures and string functions

## Notes

- Run the scripts in sequence for the best result.
- The later files depend on the database and tables created by the earlier files.
- These scripts are meant to be run in MySQL Workbench or the MySQL command line.
